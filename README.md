#
■概要：<br/>
「Haroteku」はandroidstudioを使用してJavaで制作したゲームアプリです。<br/>
androidタブレットでのプレイを想定し、画面上から落下してくる障害物を避けながら進むシンプルな作りのゲームです。
スタート画面から「会話パート」「アクションパート」を繰り返してステージが進んでゆき、
3ステージ＋エキストラモードで全４ステージを実装しています。

■アプリ説明：Haroteku（言語java）：製作期間およそ3週間<br/>
本アプリはandroidの部品である「シークバー」に着目し、
「これを使えば感覚的なゲームができるのでは」という発想から開発が始まりました。<br/>
これを利用するために（ここから始める）
フォルダ丸ごと入れるとandroidstudioの設定がまるまる入ってしまうので、<br/>
記述した「ソースコード」と「画面設計などその他」とを抜粋して掲載しました。<br/>
