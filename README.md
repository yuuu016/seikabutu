#
■概要<br/>
「Haroteku」はandroidstudioで制作したゲームアプリです。
「応募フォーム」はインターンで制作した応募フォームを思い出しながら制作したものです。

■説明：Haroteku：製作期間およそ3週間
アンドロイドタブレットで遊ぶことを想定した落ちものを躱すゲーム
フォルダ丸ごと入れるとandroidstudioの設定がまるまる入ってしまうので、
記述した「ソースコード」と「画面設計などその他」と抜粋して掲載しました。
苦労したのは
ーーーー
「各ステージの切り替え処理（Stageset.javaとTalkset.java）」
　会話終了で画面切り替えをする処理やゲームクリアのタイミングで画面転移など
  イメージはできていても実装するとどうやるかわからなかった。
「当たり判定処理（Hurdle.java）」
  学校で習っていないことの上、参考書にも書かれていないことだったので、
  ネットで調べながら手探りで実装した。
ーーーー
になります。

■説明：応募フォーム：製作期間およそ１週間
インターン中に制作した応募フォームを再び作ってみました。
納品会で言われたことを踏まえつつ、学んだことの振り返りも兼ねて制作をしました。
まだ粗く脆弱性見える甘い作りになっているかと思いますが、
インターンで想定された完成度までは完成できたと思っています。
苦労したのは
ーーーー
  「DBの設定（そもそもDBの設定ミスに気が付くのに時間がかかった）」
  「PHPの記述ミス（値を呼び出したいのに＄['文字列']で文字列そのものを呼び出していた）」
ーーーー
などです。再び作っても１週間近くかかってしまったので、できれば３日で完成させたかったです。
