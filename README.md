「Haroteku」はandroidstudioで制作したゲームアプリです。<br/>
「応募フォーム」はインターンで制作した応募フォームを思い出しながら制作したものです。<br/>

アンドロイドタブレットで遊ぶことを想定した落ちものを躱すゲーム<br/>
フォルダ丸ごと入れるとandroidstudioの設定がまるまる入ってしまうので、<br/>
記述した「ソースコード」と「画面設計などその他」と抜粋して掲載しました。<br/>
苦労したのは<br/>
ーーーー<br/>
「各ステージの切り替え処理（Stageset.javaとTalkset.java）」<br/>
　会話終了で画面切り替えをする処理やゲームクリアのタイミングで画面転移など<br/>
  イメージはできていても実装するにはどうやるかわからなかった。<br/>
「当たり判定処理（Hurdle.java）」<br/>
  学校で習っていないことの上、参考書にも書かれていないことだったので、<br/>
  ネットで調べながら手探りで実装した。<br/>

になります。<br/>

■説明：応募フォーム：製作期間およそ１週間<br/>
インターン中に制作した応募フォームを再び作ってみました。<br/>
納品会で言われたことを踏まえつつ、学んだことの振り返りも兼ねて制作をしました。<br/>
まだ粗く脆弱性見える甘い作りになっているかと思いますが、<br/>
インターンで想定された完成度までは完成できたと思っています。<br/>
苦労したのは<br/>
ーーーー<br/>
  「DBの設定」そもそもDBの設定ミスに気が付くのに時間がかかった<br/>

  「考えることが多いこと」DBの連携、DBの記述、変数と値の受け渡し、PHPとHTMLの記述の違い…など、コードを書くうえで考慮することが多く、必要な手順も複数あるのが大変だった。<br/>
ーーーー<br/>
などです。再び作っても１週間近くかかってしまったので、できれば３日で完成させたかったです。<br/>
